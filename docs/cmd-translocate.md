# /translocate

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/translocate [<playername>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Translocates your target to their bind. If a player name is specified, it will translocate that player to their bind. e.g. `/translocate Redbot`
<!--cmd-desc-end-->
